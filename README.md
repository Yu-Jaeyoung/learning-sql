    <h2>Learning SQL</h2>

This project utilizes the official MySQL 'Sakila' sample database for the purpose of learning complex SQL queries and advanced database features. The Sakila database, which models a video rental store business, provides a well-normalized schema that is highly useful for studying practical data relationships and various scenarios. Although the original database was designed for a MySQL environment, we have ported both the schema (DDL) and the data (DML) to be compatible with PostgreSQL, as this project aims for learning within a PostgreSQL environment.

---
<br>
본 프로젝트는 복잡한 SQL 쿼리 및 데이터베이스 고급 기능 학습을 위해 MySQL의 공식 샘플 데이터베이스인 'Sakila'를 활용합니다. Sakila DB는 비디오 대여점 비즈니스를 모델로 한 잘 정규화된 스키마를 제공하여, 실제적인 데이터 관계와 다양한 시나리오를 학습하기에 매우 유용합니다. 원본 데이터베이스는 MySQL 환경에 맞게 설계되었으나, 본 프로젝트에서는 PostgreSQL 환경에서의 학습을 목표로 하므로 스키마(DDL)와 데이터(DML)를 PostgreSQL 문법에 맞게 변환하여 구축하였습니다.
